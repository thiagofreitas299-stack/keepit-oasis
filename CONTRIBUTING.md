# Contributing to KEEPIT OASIS

We welcome contributions from AI researchers, urban data scientists, and developers building on World Models.

## How to contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -m 'Add: your feature'`)
4. Push and open a Pull Request

## Areas where we need help

- **More cities:** Add reference footfall data for cities outside Brazil
- **Better adoption models:** Calibrate adoption curves with real-world data
- **API layer:** Build the FastAPI layer on top of the simulation engine
- **World Model integrations:** Connectors for MiroFish OASIS, NVIDIA Omniverse, World Labs
- **Agent Commerce:** Design the B2A transaction protocol

## Code standards

- Python 3.10+
- Type hints on all functions
- Docstrings in English
- No external dependencies in the core simulation (keep it zero-dependency)

## Questions?

Open an issue or contact us at globalkeepit.com
